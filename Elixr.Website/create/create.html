Creating a character

The first step to creating your character is to either print a Character Sheet, or save one on Elixr.

A Character Sheet is used to record information and game stats about your character.

Once you have a Character Sheet ready, follow these steps to create your character and record the information.

  1. Record base stats
  2. Describe your Character
  3. Pick a Race
  4. Determine Abilities
  5. Gain Skills
  6. Choose Special Features
  7. (Optional) Select Flaws
  8. (Optional) Take Oaths
  9. Gear Up
  10. Start Playing
